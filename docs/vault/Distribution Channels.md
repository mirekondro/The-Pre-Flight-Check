# Distribution Channels

Five ways to install, all shipped by one [[Release Pipeline]].

- **PyPI / pipx** — `pipx install pre-flight-check`
- **Homebrew** — `brew tap mirekondro/pre-flight-check` (via the
  [[Homebrew Tap]]) then `brew install pre-flight-check`
- **Scoop** — `bucket/pre-flight-check.json` in this repo
- **Claude plugin marketplace** — `.claude-plugin/`
- **curl / PowerShell** — `install.sh` / `install.ps1`

Versions and checksums for Homebrew + Scoop are bumped automatically by the
[[Release Pipeline]]. Installed CLI = [[CLI]].
